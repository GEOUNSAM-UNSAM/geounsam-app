# GeoUNSAM — Waze de Aulas UNSAM

App web colaborativa para localizar aulas y recibir alertas 
de cambios en tiempo real en el campus de la UNSAM.

## Stack
- React + Vite + Tailwind CSS
- Supabase (PostgreSQL + Realtime + Auth)
- Leaflet.js (mapa del campus)
- SVG interactivo (mapa interno Tornavias)

## Setup local

1. Clonar el repo
2. npm install
3. Crear .env.local con las variables de Supabase
4. npm run dev
